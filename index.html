CampusConnect

Flutter mobile app compatible with iOS, Android and web

For developers

To run build runner

flutter pub run build_runner build --delete-conflicting-outputs

The above will generate router and locator files

To generate documentation

flutter pub global run dartdoc:dartdoc

To view documentation

pub global activate dhttpd
pub global run dhttpd --path doc/api

Navigate to http://localhost:8080 in your browser

To generate tree structure

Use file-tree-generator extension in VSCode
